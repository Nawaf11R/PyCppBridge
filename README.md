
PyCppBridge demonstrates real-time communication between Python (Pygame) and a C++ physics engine using shared memory.

In this project:
- Pygame handles rendering and input
- C++ handles physics simulation
- Shared memory is used for ultra-fast data exchange
- Shared memory (low latency, zero-copy)
